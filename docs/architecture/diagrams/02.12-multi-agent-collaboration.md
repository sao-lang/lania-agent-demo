## 概述

多 Agent 协作链路对应 §八 案例二（文档分析）和案例三（Coding Agent），涉及以下关键组件：

| 组件 | 职责 | 状态 |
|---|---|---|
| **AgentDefSubAgentAdapter** | AgentDefinition → SubAgentDef 适配 | 已设计 |
| **SubAgentDef** | 子 Agent 运行时定义（独立配置工具白名单/system prompt/budget） | 已设计 |
| **CustomizationEngine._sub_agent_registry** | 子 Agent 注册表 | 已设计 |
| **AgentOrchestrator** | 管理子 Agent 生命周期（spawn / spawn_parallel） | 已设计 |
| **TaskDecomposer** | 复杂任务分解为 3-5 个可并行/串行子任务 | 已设计 |
| **ApprovalWorkflow** | 流程级别审批节点（条件触发/指定审批人/超时拒绝） | 已规划 |
| **BudgetPool (D23)** | 全局预算计数器 | 已设计 |
| **AgentBus (D17)** | Agent 间消息总线（request_reply / pub_sub / negotiate） | 已设计 |
